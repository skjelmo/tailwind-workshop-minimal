Utility first.

Utility - just a fancy word for a class name.

“I’ve written a few thousand words on why traditional “semantic class names” are the reason CSS is hard to maintain, but the truth is you’re never going to believe me until you actually try it. If you can suppress the urge to retch long enough to give it a chance, I really think you'll wonder how you ever worked with CSS any other way.”

Adam Wathan
Creator of Tailwind CSS

Use existing utilities.

Tailwind has utilities (class names) for almost any CSS property you will use. This page is styled purely with Tailwind utilities:

  • Tailwind's font size, text color and font weight utilities are used to style the cards text (text-xl, text-indigo-500 and font-bold).
  • Tailwind's width and height utilities are used to set the height of the profile image above (w-10, h-10).
  • Tailwind's padding utilties are used on the cards to add spacing in different directions (py-4, px-6).
  • Tailwind's margin utilties are used to add space between the cards and center them horizontally (mt-6, mx-auto).
  • Tailwind's flexbox utilities are used in the quote card's footer above (flex, items-center).
  • Tailwind's max width utilities are used to constraint the width of the quote text above (max-w-3xl).
